# Celestial Obsidian UI

Автономный HTML-прототип развития направления `Obsidian`.

Открыть: `newdesign/index.html`.

В этой итерации:

- окно можно перетаскивать за верхнюю панель или бренд;
- окно можно растягивать за нижний правый угол;
- кнопки в правом верхнем углу эмулируют compact, maximize и reset;
- кнопка слева в titlebar сворачивает боковую навигацию;
- вкладки наполнены по структуре текущих страниц приложения;
- простой режим намеренно не включен.

Покрытые разделы:

- Главная: профиль, текущий узел, TUN/system proxy, трафик, runtime summary.
- Прокси: rule/global/direct, provider, chain mode, фильтры, выбранный узел.
- Профили: URL import, batch mode, runtime config, update/edit/delete actions.
- Соединения: active/closed, table/list action, сортировка, фильтр, process groups, detail.
- Правила: provider refresh, поиск, line-number list.
- Логи: фильтр уровней и поток событий.
- Уведомления: empty/feed state.
- Настройки: System, Clash, Verge Basic, Verge Advanced.
